# Интерфейс управления транками VSS AUTO HUB

## Описание

Интерфейс управления транками для личного кабинета проекта VSS. Предоставляет полный функционал для управления транками, слотами, звонками и доступа к устройствам.

## Файлы

- `public/project-trunk-dashboard.html` - HTML интерфейс
- `public/project-trunk-dashboard.js` - JavaScript логика

## Доступ

Интерфейс доступен по адресу:
```
http://localhost:3000/project-trunk-dashboard.html?projectId=<ID_ПРОЕКТА>
```

Или через кнопку "Управление транками" в списке проектов на странице Dashboard.

## Функционал

### Управление транками
- Просмотр списка транков (audio-host)
- Добавление новых транков
- Тестирование транков
- Перезапуск транков
- Удаление транков

### Управление слотами
- Просмотр статуса слотов (активен/неактивен/предупреждение/занят)
- Доступ к слотам через CLI/SSH
- Доступ к слотам через Guacamole RDP
- War Room для мониторинга

### Управление звонками
- Просмотр активных звонков
- Создание новых звонков
- Завершение звонков
- Перевод звонков
- Быстрый вызов

### Мониторинг
- Статистика CPU, RAM, сети
- Системные логи
- Статус транков в реальном времени
- Статистика звонков

### Доступ к устройствам
- **CLI/SSH**: Командная строка через SSH для ADB, терминала, файлов
- **Guacamole RDP**: Графический интерфейс через VNC/RDP

## Интеграция

Интерфейс автоматически загружает данные проекта из API:
- `GET /api/projects/:id/info` - получение информации о проекте

## Использование

1. Откройте Dashboard проектов: `http://localhost:3000/dashboard.html`
2. Выберите проект из списка
3. Нажмите кнопку "Управление транками"
4. Интерфейс откроется с данными выбранного проекта

## Особенности

- Адаптивный дизайн (мобильные, планшеты, десктопы)
- Мобильное меню для удобной навигации
- Real-time обновление статистики
- Модальные окна для различных действий
- Интеграция с Tailscale для сетевого доступа
